🎨 واجهة Session Links الجديدة:

  • نافذة منبثقة مخصصة: لإدارة روابط كل جلسة على حدة
  • 4 أنواع روابط:
    • 📄 Résumé - ملخص الجلسة
    • 📚 Support - مواد الدعم
    • 🎥 Vidéo - فيديوهات الجلسة
    • ✏️ Exercice - تمارين وأنشطة
  • عنوان مخصص: إمكانية إضافة عنوان مخصص لكل رابط
  • معاينة الروابط: عرض جميع الروابط مع إمكانية الحذف
  • أيقونات مميزة: كل نوع له أيقونة مميزة

📋 خطوات الاستخدام:

  1. إنشاء Formation: أضف تكوين جديد
  2. إضافة Courses: اضغط "Cours" → أضف كورسات
  3. إضافة Sessions: اضغط "Sessions" → أضف جلسات للكورس
  4. إدارة Links: اضغط "Liens" بجانب الجلسة → أضف الروابط الأربعة
  5. حفظ البيانات: احفظ المشارك لحفظ جميع البيانات

⚠️ ملاحظات مهمة:

  • الترتيب مهم: يجب إنشاء Formation → Course → Session → Links بالترتيب
  • الحفظ: البيانات تحفظ في localStorage حتى يتم ربطها بالـ Backend
  • الروابط الفارغة: كل جلسة تبدأ بـ links فارغة، يمكن إضافة الروابط لاحقاً
  • التحقق: تأكد من صحة URLs قبل الحفظ

interface SessionLink { id: string; type: "Résumé" | "Support" | "Vidéo" | "Exercice"; title: string; url: string; } interface Session { id: string; title: string; description: string; duration: string; isCompleted: boolean; order: number; links: SessionLink[]; // ✅ Updated } interface Course { id: string; title: string; description: string; progress: number; isCompleted: boolean; duration: string; modules: Module[]; sessions: Session[]; // ✅ Contains SessionLinks }
